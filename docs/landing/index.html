Compiles straight to WebAssembly components

Every program has exactly one form.

Canon is a maximally opinionated programming language. No let, no if, no comments, no local variables. Wherever ordering is discretionary, the compiler enforces alphabetical order.

The rules

Small surface. Sharp edges.

Style is not a choice, so a diff only ever shows meaning. What most languages leave to taste, Canon decides once, in the compiler.

a→z

Alphabetical, enforced

Product fields, union variants, functions, dispatch arms, imports: everything sorts. canon check --fix does it; unsorted source is a compile error.

*(…)

Branching is dispatch

There is no if. A union is a set of cases and a dispatch handles all of them. Forgetting one is a type error, not a 3 a.m. page.

.wasm

WebAssembly, natively

Every build is a standard WASI P3 component. No external toolchain, no transpile step. canon run executes it on embedded wasmtime.

(A * B)

No privileged receiver

Functions are defined over compositions of types. a.f(b) and b.f(a) are the same call; composition is commutative.

-> R

Effects are capabilities

A function that touches the world says so in its signature. Effects are passed in as values. Purity is the default, not a discipline.

§ 1

One-based, on purpose

Indexing starts at 1 and ranges include both ends, everywhere. The convention you explain to no one.

The web

A server is a function.

Canon picks the entry point by its type. Define Request => Response and the compiler emits a standard wasi:http/service component. Routing is dispatch on the path.

Routes without a router

Literal dispatch branches on the request path; the required catch-all arm means an unhandled route can’t exist. The same program serves locally and runs on any component host.

$ canon run notes.can
HTTP handler detected: serving on http://127.0.0.1:8080
notes.can 
Request => Response {
    Request.path() -> (
        * None => Response { NotFound() }
        * Some<String> => Response {
            String -> (
                * "/notes" => Response { Index() }
                * String => Response { NotFound() }
            )
        }
    )
}
